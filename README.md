# whatsmeow

Local workspace for WhatsApp automation.

## Projects

- [`ai-receptionist/`](ai-receptionist/) — WhatsApp AI receptionist / personal auto-reply (Go + whatsmeow)

## Git

This folder is a local git repo. Work inside `ai-receptionist/` as usual, then commit from the repo root:

```bash
git status
git add ai-receptionist/
git commit -m "Describe your change"
```

Secrets (`.env`, `*.db`) are gitignored.
